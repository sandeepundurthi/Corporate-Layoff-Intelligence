# 📉 Corporate Layoff Intelligence Platform

An executive-style workforce analytics and business intelligence platform that analyzes global corporate layoff trends across industries, countries, company stages, and funding levels.

Built using Python, Pandas, Plotly, and Streamlit.

---

# 🚀 Project Overview

This project transforms real-world corporate layoff data into actionable business insights through interactive dashboards, workforce risk analysis, and economic trend exploration.

The platform helps analyze:

- Workforce instability trends
- Industry-level restructuring patterns
- Geographic layoff concentration
- Funding-stage workforce risk
- Economic volatility over time
- Layoff severity across companies

# 📈 Business Insights Generated

- Technology and consumer industries experienced significant restructuring activity.
- Highly funded firms still conducted major workforce reductions.
- Workforce volatility aligned with periods of economic uncertainty.
- Some smaller companies experienced extremely severe workforce reductions relative to scale.

---

# 🧠 Custom Analytics Metrics

## Layoff Severity Score
A custom metric combining:
- layoff percentage
- total employees laid off

to estimate workforce restructuring intensity.

## Industry Instability Analysis
Measures industry risk using:
- total layoffs
- affected companies
- average workforce reduction

# ▶️ Run the Project Locally

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Corporate-Layoff-Intelligence.git
```

## Navigate to Project

```bash
cd Corporate-Layoff-Intelligence
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Streamlit Dashboard

```bash
cd dashboard
streamlit run app.py
```
